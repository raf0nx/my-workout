## Local Development

To run `my-workout` locally, follow these steps:

1. Clone the repo:

```bash
git clone git@github.com:raf0nx/my-workout.git
```

2. Install dependencies:

```bash
yarn install
```

3. Launch the development server:

```bash
yarn dev
```

4. Launch the Firestore emulator:

```bash
firebase emulators:start
```
